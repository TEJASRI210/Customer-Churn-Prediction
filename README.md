# Customer Churn Prediction

## Overview
Built an end-to-end Machine Learning model to predict whether a telecom customer is likely to churn.

## Dataset
- Telco Customer Churn Dataset
- Features: Gender, Tenure, Monthly Charges, Contract, Payment Method, etc.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Workflow
- Data Collection
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Label Encoding
- Train-Test Split
- Random Forest Model
- Model Evaluation
- Customer Churn Prediction

## Results
- Built a Random Forest classifier to predict customer churn.
- Evaluated the model using Accuracy, Confusion Matrix, and Classification Report.

## Repository Structure
- `Customer_Churn_Predication_using_ML.ipynb` – Jupyter Notebook
- `WA_Fn-UseC_-Telco-Customer-Churn.csv` – Dataset
- `README.md` – Project documentation
